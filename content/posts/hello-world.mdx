This is the first post on my new site. I built it because I wanted one place
that's actually mine — not a feed, not a platform that changes its rules every
six months, just a corner of the web I control.

## What you'll find here

I'm planning to write about three kinds of things:

- **Build logs** — what I'm making and the decisions behind it.
- **Things I learned the hard way** — bugs, postmortems, and the occasional
  "why did nobody tell me this sooner."
- **Half-formed ideas** — the stuff that isn't ready for a polished essay but
  is worth thinking out loud about.

## A quick note on the format

These posts are written in MDX, which means I can drop interactive components
right into the prose when a static paragraph won't cut it. Code samples render
nicely too:

```ts
function greet(name: string): string {
  return `Hello, ${name}`;
}
```
